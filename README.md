# VibeCoffee

A simple coffee ordering web app where users can browse a menu, customize a drink, add it to an order, and choose an in-store pickup location.

## Files

- `index.html` — main UI and layout
- `styles.css` — styling for the app
- `app.js` — ordering logic and state management

## How to run

Open `index.html` in a browser.

## Features

- Menu with drink cards
- Customizable drink options: size, milk, add-ons, quantity
- Order summary with remove buttons
- Pickup location and time selection
- Confirm order with pickup status message
